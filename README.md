# ALE2GBK-Converter
用于将AvidLogExchange（ALE）文件转换为GBK编码得以让AVID Media Composer支持中文元数据导入

---

中文 | [English](./README_en.md) 


## 如何使用
#### 方法一
```console
python ALE2GBK-Converter.py /path/to/your/input_file.ale
```
